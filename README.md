### Bright Coffee Shop Sales – Data Analytics Project

## Project Overview
Analysis for new COE to undersyand the sales at bright coffee shop sales for last 6 months.


### Objectives
The main goals of this project are:
• Identify which products generate the highest revenue.
• Detect seasonal trends in product performance.
• Provide recommendations to improve overall sales performance.

### Tools & Technologies

SQL for querying and analyzing raw transactional data.
Databricks for running queries and scripts.
Microsoft Excel and Powe BI for Data Visualization 
PowerPoint for Presentation and Reporting.
